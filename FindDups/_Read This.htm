1) What does the duplicates finder do?

  

-The duplicates finder helps you to save a great amount of your hard disk by comparing files’ data, it doesn’t match the same only but also it gives many degrees of data similarities starts from 1% similarity to 100%

-100% means no changes in the full file, so all files will be displayed

-I want to tell you that you must know the following before using the program:

 a- the program compares each file in the directory you choose with the other so if you gives the program the root directory of the hard disk it will take a lot of time to finish it’s task, so I want to tell you, you must choose directories that may contain files data may be the same for example:

   you have a hard drive root D:\ contains media files

            inside this directory there are

            -Photo         folder contains photos files

            -Mp3           folder contains Mp3 songs

            -Midi           folder contains Midi Music Files

  it will take a lot of wasted time if you choose to compare all the D:\ in one time, but what you should do to compare Photo, then Mp3, Then Midi

I think I will free up a huge space from your data
